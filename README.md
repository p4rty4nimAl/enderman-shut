# Enderman Sound Adjuster

A Minecraft Fabric mod with two main features:  

One feature seperates enderman sounds into another slider, allowing their volume to be changed relative to the default "Hostile Creatures". This is slider is seen highlighted in the following image:  
  ![Image highlighting a slider: Enderman Screams](https://p4rty.4nimal.tk/projects/esa/esa-slider-demo.png "Enderman Screams slider")
  
  
The other feature this mod introduces is the ability to replace the screams of hostile endermen with their passive counterparts, as highlighted in the image below:  
  ![Image highlighting a button: Ambient Screams](https://p4rty.4nimal.tk/projects/esa/esa-button-demo.png "Ambient Screams button")
